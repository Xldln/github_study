1. --创建自己的github仓库
-- 自行按照引导 完成建库
2.--尝试上传一个本地文件
--git bash 打开本地文件夹
3. --git init 来创建初始化这个项目
--使用main 替换原来的master的项目名称 原因是master政治不恰当
 -- git branch -m main 进行替换主干名称
 -- git status  展示当前git要上传的项目的状态信息
4. -- git add -A 跟踪所有文件的改变
 文件变为绿色则说明完成追踪
5.git commit -m "my_commit" 用来添加本次更新内容
通过以上操作，就已经把这次新的节点的内容准备好了，下面需要将其上传到github仓库中
6.设置ssh密钥
设置密钥比较麻烦
需要到官网的ssh申请并设置密钥，在随意一个文件下或者你要上传的文件夹下边 使用git bash打开
然后需要输入ssh-keygen -t ed25519 -C "youremail@example.com"
随后自己创建passphrase密码，得到一个密钥
随后在路径下边创建并打开
torch ~/.ssh/config
nano ~/.ssh/config
打开以后在文件中输入内容
Host *
  AddKeysToAgent yes
  UseKeychain yes
  IdentityFile ~/.ssh/id_ed25519
ctrl+o保存 ctrl+x退出
cat ~/.ssh/id_ed25519.pub打印出公共密钥
将得到的ssh密钥放到github中 然后连接自己的电脑
ssh —T git@github.com
最后连接完毕
可以开始上传文件了
7.上传文件需要先设置好origin
使用git remote add origin https://github.com/****/****.git设置好自己仓库的url
然后使用git push -u origin main 完成推送
